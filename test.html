خب اولین مورد مثلا استفاده از لینک بدون پسوند http مناسب نیست.

<a href="example.com">My Website Link</a>

بجای قطعه کد بالا از کد زیر استفاده کنیم :

<a href="https://example.com">My Website Link</a>

اگر صفحه مربوط به لینک ما برای کشور یا منطقه خاصی است میتوانیم عنوان کشور را در لینک خود قرار دهیم. مثال :

<a href="https://example.ir">My Website Link</a>
<a href="https://example.com/ir/">My Website Link</a>

مورد بعد اینه که داخل لینک کلمات رو از هم جدا کنیم مثلا لینک زیر اصلا مناسب نیست.

<a href="https://example.ir/hometools">home tools</a>

باید واژگان رو در لینک از هم جدا کنیم. دقت کنیم هرگز در لینک یا تصاویر واژگان رو با علامت _ یا فاصله از هم جدا نکنیم میتوان از علامت - یا / استفاده کرد.

<a href="https://example.ir/home/tools">home tools</a>
<a href="https://example.ir/home-tools">home tools</a>

مورد بدتری که در برخی صفحات انجام میشود ساخت لینکها بوسیله id در قالب درخواست get است مانند نمونه :

<a href="https://example.ir/post?id=1234">home tools</a>

بجای این مورد میتوان از عنوان لینک در ساخت لینکها کمک گرفت. دقت کنید در جداسازی کلمات در لینکها از کاراکتر _ استفاده نشود.

ویژگی rel در تگ لینک : این ویژگی چند مقدار ثابت میگیرد. اگر میخواهید به طور کامل لینک شما توسط گوگل دنبال شود. از این ويژگی استفاده نکنید.

لینکهایی که مربوط به اسپانسر صفحه ما باشند را با مقدار sponsored برای ويژگی rel مشخص میکنیم.
